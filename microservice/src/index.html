This microservice exposes the functionality of the @iqb/responses library through a REST API, enabling coding, validation, and text rendering operations for educational assessment responses.

üîå API Endpoints

Coding Operations

CodingFactory
POST /codings/code

Code a single response using a specific coding definition.

Request Body:
{
  "response": {
    "id": "var1",
    "value": "hello",
    "status": "VALUE_CHANGED"
  },
  "coding": {
    "id": "var1",
    "codes": []
  }
}

Coding Scheme Operations

CodingSchemeFactory
POST /schemes/code

Code a set of responses using a complete variable coding scheme.

Request Body:
{
  "unitResponses": [...],
  "variableCodings": [...]
}
POST /schemes/validate

Validate a coding scheme against a set of base variables.

Request Body:
{
  "baseVariables": [...],
  "variableCodings": [...]
}
POST /schemes/derive-value

Derive a value for a variable based on other variable values (derivation rule).

Request Body:
{
  "variableCodings": [...],
  "coding": {...},
  "sourceResponses": [...]
}

Text Rendering Operations

ToTextFactory
POST /text/code

Generate a text description for a specific code rule.

Request Body:
{
  "code": {...},
  "mode": "EXTENDED"  // Optional
}
POST /text/source

Generate a text description for a source definition.

Request Body:
{
  "variableId": "var1",
  "sourceType": "BASE",
  "sources": [...],
  "parameters": {...}  // Optional
}
POST /text/processing

Generate a text description for processing instructions.

Request Body:
{
  "processing": [...],
  "fragmenting": {...}  // Optional
}
POST /text/var-info

Generate a text description for general variable information.

Request Body:
{
  "varInfo": {...}
}

üí° Example Usage

Using fetch (JavaScript/TypeScript)

const response = await fetch('http://localhost:3000/codings/code', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    response: { id: 'var1', value: 'hello', status: 'VALUE_CHANGED' },
    coding: { id: 'var1', codes: [] }
  })
});
const result = await response.json();
console.log(result);

Using curl

curl -X POST http://localhost:3000/codings/code \
  -H "Content-Type: application/json" \
  -d '{
    "response": {"id": "var1", "value": "hello", "status": "VALUE_CHANGED"},
    "coding": {"id": "var1", "codes": []}
  }'

üê≥ Docker Integration

This service is designed to run as a Docker container and can be integrated into your application stack:

version: '3.8'

services:
  responses-service:
    build:
      context: ./microservice
      dockerfile: Dockerfile
    ports:
      - "3000:3000"
    
  my-app:
    image: my-app-image
    depends_on:
      - responses-service
    environment:
      - RESPONSES_SERVICE_URL=http://responses-service:3000
